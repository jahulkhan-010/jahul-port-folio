## 💡 How It Works

The chatbot now uses:
1. **Text Normalization** - Removes extra spaces
2. **Lowercase Matching** - Case-insensitive
3. **Pattern Matching** - Checks multiple variations
4. **Common Typo Recognition** - Handles common mistakes
5. **Fuzzy Keyword Detection** - Matches similar terms
