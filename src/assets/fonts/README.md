# Font Files

This directory contains the Poppins font files in WOFF2 format.

You need to download the following files and place them in this directory:
- Poppins-Light.woff2
- Poppins-Regular.woff2
- Poppins-Medium.woff2
- Poppins-SemiBold.woff2
- Poppins-Bold.woff2

You can download these files from Google Fonts or a font converter service.
